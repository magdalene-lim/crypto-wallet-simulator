RP Coin Wallet Simulator

A browser-based simulated crypto wallet for learning — no real funds involved.

📚 EDUCATIONAL SIMULATOR ONLY
This is a learning tool for educational purposes. It does NOT use real blockchain technology or real cryptocurrency. Do not use generated keys for actual crypto assets.

📖 Quick Start Guide

  1. Select a Wallet: Choose between Wallet 1 or Wallet 2 below. You can rename them!
  2. Generate Wallet: Click "Generate Wallet" to create a simulated address and private key.
  3. Receive RP Coins: Click "Receive 100 RP (Faucet)" to add test funds to your wallet.
  4. Send RP Coins: Enter a recipient address and amount, then click "Send" to transfer coins.
  5. View History: All transactions are recorded in the Transaction History section below.

💡 Tip: Try creating both wallets and sending coins between them to see how transfers work!
